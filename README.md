# Daily Sales Report Automation (Python)

This project is a simple Python automation script that reads daily sales data from a text file, processes it, and generates a summarized sales report.

## What it does
- Reads sales data from a `sales.txt` file
- Calculates total revenue
- Generates item-wise quantity and revenue summary
- Identifies the best-selling item
- Saves the final report to `daily_report.txt`

## Why this project
This project was built to practice core Python concepts such as,
- File handling
- Loops and conditionals
- Functions
- Lists and dictionaries
- Basic error handling

## Input Format
Each line in `sales.txt` should follow this format - 
item_name,quantity,price_per_item


## Output
- Console summary
- A formatted report saved as `daily_report.txt`

## Tech Stack
- Python (no external libraries)
